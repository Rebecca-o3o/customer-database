## Overview
A Single Page Application which lists customer data.
Hosted on [Heroku](https://stormy-harbor-82740.herokuapp.com/)

## Technologies
- React.js
- Node.js
- Express.js
- PostgreSQL
- Bootstrap (no jQuery)


## Details

### Overview page
- Lists all customers
- Link to ‘Customer Detail’ page)
- Delete a customers
